Antes de comenzar

Este sistema registra proyectos, las instituciones que participan en ellos y los actores involucrados. Sigue el orden de las pestañas para evitar errores: primero instituciones, luego proyectos, al final actores.

1 Institución

¿Qué es? Una organización, departamento o grupo que participa en proyectos. Puede tener hasta 3 niveles jerárquicos: Organización › Departamento › Grupo.

Qué preparar: nombre oficial, tipo (ONG, gobierno, academia, etc.), alcaldía y colonia donde opera.

Ubicación: haz clic en el mapa para capturar las coordenadas exactas; si no la conoces, puedes dejarla vacía.

Jerarquía: si tu institución es un Departamento o Grupo, el formulario te pedirá los datos del nivel superior en la misma operación.

2 Proyecto

¿Qué es? Una iniciativa o acción concreta. Se vincula con las instituciones que ya registraste y con los actores que participan en él.

Qué preparar: nombre del proyecto, objetivo principal, ubicación geográfica e instituciones participantes con su rol (financiadora, ejecutora, aliada…).

Requisito: las instituciones deben estar registradas en la pestaña anterior antes de poder vincularlas aquí.

⚠ Importante: al presionar guardar proyecto, los datos se almacenan en la base de datos. Esta interfaz no permite editar ni eliminar registros, así que revisa bien la información antes de guardar.
3 Actor

¿Qué es? Una persona de contacto vinculada a un proyecto (no a una institución directamente).

Qué preparar: nombre completo, correo electrónico (obligatorio), teléfono o WhatsApp y cargo o posición dentro del proyecto.

Requisito: el proyecto al que pertenece debe estar registrado primero.

Proyecto registrado

✓

El proyecto se guardó correctamente en la base de datos.

Siguiente paso: abre el mapa para verificar que el proyecto aparece en la ubicación correcta y con sus instituciones vinculadas.

⚠ ¿Cometiste un error? Esta interfaz no permite editar ni eliminar registros. Si necesitas corregir algún dato, comunícalo al administrador del sistema para que realice el ajuste directamente en la base de datos.
